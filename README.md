## ReactJS Emoji Search

A simple ReactJS application to search in large data of emojis by keywords.

<br>

Features:
- Responsive.
- Simple code & design (good for beginering who are trying to learn ReactJS).
- SweetAlert2.

How To:
- Download or clone repo.
- run `npm install`.
- run `npm start`.
